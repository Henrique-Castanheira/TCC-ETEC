📋 Sobre a Choki
A Choki nasce com o propósito de facilitar o acesso a dados públicos e simplificar o entendimento sobre o consumo elétrico. Nosso foco é transformar números complexos em decisões práticas para evitar o desperdício e promover a sustentabilidade.

O projeto consiste no desenvolvimento de um protótipo de simulador energético que permite estimar o consumo mensal de uma residência e simular o impacto financeiro de mudanças de hábito e novos equipamentos.

🎯 Diferenciais do Nosso Sistema
Apoio à Decisão: Diferente de um medidor comum, a Choki oferece uma ferramenta analítica fundamentada em dados oficiais.

Simulação de Impacto: Veja a economia real antes mesmo de trocar uma lâmpada ou eletrodoméstico.

Integração Hardware/Software: Utilizamos a precisão do sensor SCT-013 com o processamento do ESP32 para entregar dados confiáveis via LCD 16x2.

🛠️ Stack Tecnológica
Microcontrolador: ESP32

Sensores: SCT-013 (Captação de corrente)

Display: LCD 16x2 i2C

Cálculos: Algoritmos baseados em potência média, tarifas oficiais e lógica de Payback.

📉 Indicadores de Eficiência
A ferramenta da Choki gera automaticamente:

Ranking de Intervenções: Quais trocas trazem o melhor custo-benefício.

Economia Projetada: Comparativo anual do cenário atual vs. cenário otimizado.

Tempo de Retorno: Em quanto tempo o investimento em eficiência se paga.
